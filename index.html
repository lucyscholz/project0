Featured Project

Google App Script Data Writer

  • Automates the copying of data from spreadsheet to another on a large scale.
  • Allows for data to be filtered prior to copying so only certain rows are copied.
  • Can be set to run daily to automate the updating of key spreadsheets.
  • Keeps a log so you can track what copies have occured - records date, time, and how many rows were copied.
  • Roster Update feature can update a tab in the destination sheet with a set of unique IDs to include any new rows that have been added to the dataset since the last copy.
